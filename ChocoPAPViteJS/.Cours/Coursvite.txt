Pour que l'import de path fonctionne : 
npm i @types/node -D
et rajouter "types":["node"] dans tsconfignode.json

/!\ Ne pas mettre Bootstrap car pour react, il n'y a que react bootstrap, qui passe par à la fois react et bootstrap, il risque de ne pas être maintenu comme bootstrap normal.
npm i --save bootstrap @popperjs/core
Continuer sur https://getbootstrap.com/docs/5.2/getting-started/vite/#import-bootstrap


Voir https://github.com/vitejs/vite/discussions/12191 pour l'import d'images d'un folder

Dans vite.config.ts, ajouter dans l'export : 
esbuild: {
    loader: 'tsx',
  },

Les fichiers js/ts doivent prendre l'extension tsx
